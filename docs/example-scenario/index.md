# <a name="azure-example-scenarios"></a>Azure 示例方案

这些示例方案基于真实客户示例。 它们旨在加快和简化构建出你自己的用例的过程。 每个解决方案都包含一张体系结构图，探讨了一些可以为了满足自己业务需求而替换的替代项，以及解决方案的粗略定价。  大多数解决方案还将包含用于轻松地将解决方案部署到自己 Azure 帐户中的方法。
